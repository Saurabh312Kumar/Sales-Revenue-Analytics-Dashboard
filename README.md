# Sales-Revenue-Analytics-Dashboard
# Data Analytics Project

##  Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to business insights visualization. The goal is to extract meaningful insights using Python, SQL, and Power BI, and present them in a clear, decision-friendly format.

## 📁 Dataset

* The dataset contains structured business data including:

  * Sales
  * Profit
  * Quantity
  * Customer and product details
* Source: Public dataset (e.g., Superstore dataset)

---

## 🛠️ Tools & Technologies

* **Python** – Data cleaning, preprocessing, and EDA
* **Pandas** – Data manipulation
* **MySQL** – Querying and data analysis
* **Power BI** – Dashboard creation and reporting


## Project Steps
### 1. Data Loading (Python)
* Imported dataset using Pandas
* Checked structure, data types, and missing values

### 2. Data Cleaning
* Handled missing values
* Removed duplicates
* Standardized column formats

### 3. Exploratory Data Analysis (EDA)
* Analyzed sales trends, profit distribution, and category performance
* Created visualizations to identify patterns and outliers

### 4. SQL Analysis (MySQL)
* Imported cleaned data into MySQL
* Wrote queries to answer business questions:

  * Monthly sales trends
  * Top-performing products
  * Region-wise performance
  * Profit analysis

### 5. Power BI Dashboard
* Connected Power BI with MySQL / dataset
* Built interactive dashboards with:

  * KPIs (Total Sales, Profit, Quantity)
  * Filters (Year, Region, Category)
  * Charts (Bar, Line, Pie)

## 📊 Dashboard

The Power BI dashboard provides:

* Clear visualization of key business metrics
* Interactive filtering for deeper analysis
* Insights into trends, top products, and regional performance

---

## 📈 Results & Insights

* Identified top-performing products and categories
* Analyzed seasonal sales trends
* Highlighted profit/loss areas
* Provided actionable insights for business decision-making
